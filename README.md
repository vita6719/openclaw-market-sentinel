# openclaw-market-sentinel
AI-powered Wildberries analyzer for sellers. Built with Python 3.14, Flet, and local TinyLlama/Ollama.
# 📦 OpenClaw Market Sentinel 2026

**AI-ассистент для анализа маркетплейсов на базе локальных LLM.**

Инструмент для малого бизнеса, который парсит данные Wildberries по артикулу и дает мгновенный бизнес-совет с помощью локальной нейросети.

### 🛠 Стек технологий:
- **Язык:** Python 3.14 (актуальная версия!)
- **Интерфейс:** Flet (современный Material Design)
- **ИИ-мозг:** TinyLlama / Ollama (работает полностью локально, данные селлера приватны)
- **API:** Прямая интеграция с Wildberries API

### 🚀 Особенности:
- **Приватность:** Все расчеты и советы от ИИ происходят на вашем ПК.
- **Оптимизация:** Работает даже на бюджетных устройствах благодаря выбору TinyLlama.
- **Бизнес-логика:** Расчет реальных цен и быстрый аудит карточки товара.

### 💻 Как запустить:
1. Установите [Ollama](https://ollama.com) и модель `tinyllama`.
2. Скачайте `Market_Vision.exe` из раздела Releases.
3. Введите артикул товара и получите совет от ИИ!
